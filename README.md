# 💰 Expense Tracker Flutter

A modern and user-friendly Expense Tracker application built with **Flutter** to help users manage their daily finances, track expenses, and gain insights into their spending habits.

---

## 📱 Features

- ➕ Add new income and expense records
- ✏️ Edit existing transactions
- 🗑️ Delete transactions
- 📊 Interactive expense charts
- 💵 Track total income and expenses
- 📈 Financial summary dashboard
- 💾 Local SQLite database
- 📄 Export reports as PDF
- 🎨 Clean and responsive Material UI

---

## 🚀 Getting Started

### Prerequisites

- Flutter SDK
- Android Studio or VS Code
- Android Emulator or Physical Device

### Installation

Clone the repository

```bash
git clone https://github.com/Rishi2-hub/expense_tracker_flutter.git
```

Navigate to the project

```bash
cd expense_tracker_flutter
```

Install dependencies

```bash
flutter pub get
```

Run the application

```bash
flutter run
```
